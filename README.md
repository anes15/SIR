## 🇸🇦 باللغة العربية

**SIR** هو أداة سطر أوامر (CLI) متطورة مكتوبة بلغة **++C**، تمنحك تحكماً كاملاً في مظهر أيقونات سطح المكتب من خلال التعامل مع سجل نظام ويندوز.

### ✨ المميزات الجديدة (التحديث الأخير)
*   🚀 **إزالة فورية:** حذف سهم الاختصار تماماً بضغطة زر.
*   🔄 **استعادة كاملة:** العودة للوضع الافتراضي في أي وقت.
*   ⚡ **تحديث تلقائي ذكي (Smart Refresh):** 
    *   يقوم البرنامج تلقائياً بإعادة تشغيل `explorer.exe` لضمان تطبيق التغييرات فوراً.

### 🚀 كيف يعمل؟
1. يقوم بالوصول إلى مفتاح السجل `Shell Icons` تحت `HKEY_LOCAL_MACHINE`.
2. ينفذ أوامر `taskkill` و `start` لإعادة إنعاش واجهة المستخدم دون الحاجة لإعادة تشغيل الكمبيوتر.

### ⚙️ المتطلبات
*   **نظام التشغيل:** Windows 7 / 8 / 10 / 11.
*   **الصلاحيات:** يجب التشغيل **كـ مسؤول (Administrator)**.

## 🇺🇸 English Version

**SIR** is an advanced command-line utility (CLI) built with **C++**, providing seamless control over Windows desktop icon overlays via Registry manipulation.

### ✨ Key Features (Latest Update)
*   🚀 **Instant Removal:** Completely hide the shortcut arrow overlay.
*   🔄 **Full Restoration:** Easily revert to the default Windows shortcut arrows.
*   ⚡ **Smart Auto-Refresh:** 
    *   Automatically restarts `explorer.exe` to ensure all changes are visible immediately.

### 🚀 How It Works
1. It accesses the `Shell Icons` registry key under `HKEY_LOCAL_MACHINE`.
2. It executes `taskkill` and `start` commands to refresh the shell environment without requiring a full system reboot.

### ⚙️ Requirements
*   **OS:** Windows 7 / 8 / 10 / 11.
*   **Permissions:** Must be **Run as Administrator**.
